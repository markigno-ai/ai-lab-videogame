# 🎮 AI Game Lab

Gioco creato in tempo reale durante una lezione sull'Intelligenza Artificiale, usando [Claude Code](https://claude.ai) come unico strumento di sviluppo.

**Zero righe di codice scritte a mano.** Solo linguaggio naturale.

## Come funziona

1. Parliamo con Claude Code in italiano
2. Claude Code scrive il codice
3. Il codice viene pushato su GitHub
4. Vercel pubblica automaticamente la nuova versione

## Tech stack

- HTML5 Canvas
- CSS3
- JavaScript vanilla
- Deploy: Vercel (automatico ad ogni push)
